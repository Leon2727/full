## Getting Started!!

`1. git clone https://github.com/resourceldg/full.git`

 - ### Backend install & local run

`2. cd full/backend`

`3. npm i` 

`4. npm start`

 - ### Frontend Getting Started!!

`5. cd full/frontend`

`6. npm i` 

`7. npm run dev`
